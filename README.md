# TenderLyte Website

Production-oriented Next.js + Tailwind landing page converted from the TenderLyte Stitch project.

## Stitch Analysis

Source project: `TenderLyte Immersive 3D Experience`

The strongest Stitch direction is the `Cinematic Purity` design system: deep forest green, electric lime accents, Sora display typography, Manrope body typography, glassmorphism, generous full-screen sections, product-first storytelling, and tropical atmospheric motion.

Visible Stitch screens reviewed:

- `TenderLyte | Product Showcase`
- `TenderLyte | Ingredients & Purity`
- `TenderLyte | Farm to Pack Journey`
- `TenderLyte | Lifestyle (Zen Minimalist)`
- `TenderLyte | Showcase & Contact (Synced Header)`

## Local Preview

Install dependencies, then run the Next.js dev server:

```sh
npm install
npm run dev
```

Then visit `http://localhost:3000`.

The previous static HTML/CSS/JS files are kept as reference exports. The production app entry point is `app/page.tsx`.
